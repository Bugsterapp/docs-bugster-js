## Existing Flows

Here you can find all the user flows that have been captured from your application. These flows represent real user interactions that can be converted into automated tests.

## Managing Flows

- View flow details and steps
- Edit flow configurations
- Delete unnecessary flows
- Convert flows to automated tests

## Flow Status

Flows can have different statuses:
- Active
- Archived
- Failed
- Needs Review

## Actions

Learn how to:
- Filter and search flows
- Sort by different criteria
- Export flow data
- Bulk actions on multiple flows
