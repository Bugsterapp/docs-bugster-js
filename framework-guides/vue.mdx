Follow these steps to integrate the Bugster SDK with your Vue.js application:

## 1. Read the docs

Before you begin, make sure to read through the Bugster documentation for a comprehensive understanding of the SDK and its auto-tracking capabilities.

## 2. Install Bugster using your package manager

Open your terminal and run the following command:

```bash
npm i @bugster/bugster-js
```

## 3. Create a Bugster plugin

Create a new file `src/plugins/bugsterPlugin.js` with the following content:

```javascript
import { BugsterTracker } from '@bugster/bugster-js';

const bugsterPlugin = {
  install(app, options) {
    // Initialize BugsterTracker with auto-tracking enabled
    const bugster = new BugsterTracker({
      apiKey: options.apiKey || 'bugster_lpzp8uwMLHwLygPTvaey3ftIoLtusVBT',
      endpoint: "https://i.bugster.app",
      autoTrack: true, // Enable auto-tracking
    });

    // Add the Bugster instance to the app's global properties
    app.config.globalProperties.$bugster = bugster;

    // Optional: provide the tracker for injection elsewhere
    app.provide('bugster', bugster);
  },
};

export default bugsterPlugin;
```

## 4. Initialize the plugin

In your main Vue application file (usually `main.js` or `main.ts`), initialize the Bugster plugin:

```javascript
import { createApp } from 'vue';
import App from './App.vue';
import bugsterPlugin from './plugins/bugsterPlugin';

const app = createApp(App);

app.use(bugsterPlugin, {
  apiKey: 'bugster_lpzp8uwMLHwLygPTvaey3ftIoLtusVBT',
  endpoint: "https://i.bugster.app",
});

app.mount('#app');
```

## 5. Auto-tracking in action

With auto-tracking enabled, Bugster will automatically capture various events and interactions within your Vue.js application without requiring manual tracking calls. This includes:

- Page views
- Click events
- Form submissions
- API calls
- Performance metrics
- Error logging

You don't need to add any additional code to your components for these events to be tracked. Bugster will handle it automatically.

However, if you need to add custom data or context to the auto-tracked events, you can use the Bugster instance in your components:

```vue
<script>
export default {
  mounted() {
    // Add custom context to auto-tracked events
    this.$bugster.setContext('user', {
      id: 'user123',
      plan: 'premium'
    });
  }
}
</script>
```

Or with Composition API:

```vue
<script setup>
import { inject, onMounted } from 'vue';

const bugster = inject('bugster');

onMounted(() => {
  // Add custom context to auto-tracked events
  bugster.setContext('user', {
    id: 'user123',
    plan: 'premium'
  });
});
</script>
```

That's it! You've successfully integrated the Bugster SDK with your Vue.js application, and auto-tracking is now enabled. Bugster will automatically collect data on user interactions and application performance without requiring manual tracking calls throughout your code.
