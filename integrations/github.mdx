## GitHub Integration

Integrate Bugster with GitHub to streamline your testing workflow and maintain test cases alongside your code.

## Setup

1. Install Bugster GitHub App
2. Configure repository access
3. Set up webhooks
4. Configure test automation

## Features

- Automated test runs on PRs
- Test status checks
- Issue creation for failures
- Test history in PR comments

## Configuration

```javascript
{
  "github": {
    "enabled": true,
    "repositories": ["owner/repo"],
    "autorun": {
      "onPR": true,
      "onPush": false
    }
  }
}
```
